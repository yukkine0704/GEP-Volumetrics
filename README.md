### DEPENDENCIES
Make sure to download this correctly:

- [GEP](https://github.com/OhioBob/Grannus-Expansion-Pack)
- [GPP_Volumetrics](https://github.com/yukkine0704/GPP-Volumetrics)
- [EVE Release 5 & Scatterer (paid)](https://www.patreon.com/blackrack)

### INSTALLATION

- After installing GEP, delete GEP\GEP_Configs\Clouds.cfg
- Download and install Environmental Visual Enhancements and Scatterer from blackrack's Patreon.
- Install GPP Volumetrics, if you don't have GPP by any chance, you can delete the GPP Volumetrics config folder, only textures are needed.
- Drop the Gamedata folder from GEP_Volumetrics zip into your game folder, replace if needed.
- Done!

### Extras

- Configs for BetterGrannus, Distant Object Enhancement and Friefly are included in the Extras folder.
- For installation, navigate to each folder and copy it's contents into your Gamedata folder, replace if needed.

### Bugs

- Volumetric clouds for Tautatis and Brovo are black for some reason.
- Too much red tint on Novens 2D clouds when using BetterGrannus.
